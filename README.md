## 概述

[luckydraw](https://github.com/weui/weui) 是一个为node后台提供的抽奖算法，包括“定量定员模式、人均成本模式、定量不定员模式”，更改参数，可以解决多数抽奖逻辑问题。

## 使用

- 组件的wxml结构请看`dist/example/`下的组件
- 样式文件可直接引用`dist/style/weui.wxss`，或者单独引用`dist/style/widget`下的组件的`wxss`
